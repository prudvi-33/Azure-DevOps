📌 Kube-API Server:
====================
➤ Kube-API Server is the primary management component in kubernetes. <br/>
✅️ When you run **kubectl get** command, it first reaches to Kube-API Server. <br/>
⤷ The Kube-API Server first authenticates the request and validates it. <br/>
⤷ It then retrives the data from the etcd cluster and responds back with the requested information. <br/>
