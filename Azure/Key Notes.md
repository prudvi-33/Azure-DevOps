* In Azure, every **Resource Group** is tied to exactly **one Subscription** <br/>

✅ **Key Relationship** <br/>

**Subscription** → contains Resource Groups <br/>
**Resource Group** → contains Resources (VMs, NSGs, Storage, etc.) <br/>
